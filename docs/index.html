Kings Road Traffic Feed

This project monitors vehicle speeds and traffic volume on Kings Road, M16. It uses a Raspberry Pi and computer vision to log how many cars pass by and how fast they’re actually going. Note: All speeds shown are reduced by 10% to ensure these figures are a conservative minimum.
